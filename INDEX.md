## 🔍 Key Concepts

### Data Flow
User Input → GitHub API → Data Processing → Database → Analytics → ML Models → API → Frontend Display

### Architecture Layers
1. **Frontend**: Next.js React components
2. **API**: FastAPI REST endpoints
3. **Services**: Business logic (analytics, ML)
4. **Database**: SQLAlchemy ORM with SQLite
5. **External**: GitHub GraphQL API

### Key Technologies
- **Backend**: FastAPI, SQLAlchemy, scikit-learn
- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Database**: SQLite
- **ML**: scikit-learn, XGBoost, LightGBM
